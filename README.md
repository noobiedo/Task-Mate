# Task-Mate
TASK MATE
Sistem Manajemen Tugas Berbasis Cloud

Di era digital yang serba cepat ini, mahasiswa dituntut untuk mampu mengelola berbagai tugas akademik maupun non-akademik secara efektif. Tingginya beban tugas yang diberikan, ditambah dengan aktivitas lain baik di dalam maupun di luar kampus, seringkali membuat mahasiswa mengalami kesulitan dalam mengatur waktu. Akibatnya, tidak sedikit mahasiswa yang mengalami keterlambatan dalam pengumpulan tugas.
Pada umumnya, pengelolaan tugas masih dilakukan secara konvensional, seperti menggunakan catatan manual berupa sticky notes, buku tulis, atau media pencatatan lainnya. Metode ini memiliki beberapa kelemahan, di antaranya kurang praktis, rentan terhadap kehilangan atau kelalaian, serta tidak adanya sistem pengingat otomatis yang dapat membantu mahasiswa mengingat tenggat waktu tugas.
Permasalahan tersebut menunjukkan adanya kebutuhan akan suatu sistem yang mampu membantu mahasiswa dalam mengelola tugas secara lebih terstruktur dan efisien. Sistem tersebut diharapkan dapat mencatat, mengorganisasi, serta memberikan notifikasi atau pengingat sebelum batas waktu pengumpulan tugas tiba.
Berdasarkan latar belakang tersebut, dirancang sebuah sistem berbasis digital yang diberi nama Task Mate, yang bertujuan untuk membantu mahasiswa dalam memanajemen tugas secara optimal. Sistem ini diharapkan dapat meningkatkan kedisiplinan, mengurangi keterlambatan, serta mendukung produktivitas mahasiswa dalam menjalankan aktivitas akademiknya.

Tujuan
1. Mengelola tugas secara terstruktur
2. Menyimpan data secara online dan aman
3. Memberikan pengingat / notifikasi tenggat tuga

Arsitektur Cloud (AZURE)

User → Static Web Apps → App Service → MySQL Database

Blob Storage untuk file

Azure Functions untuk reminder
